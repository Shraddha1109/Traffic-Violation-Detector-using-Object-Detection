Traffic Violation Detector Using Object detection

Traffic violation detector are effective tools to help traffic administration to monitor the traffic condition. It can detect traffic violations, such as running red lights, speeding, and vehicle retrogress in real time. A complete traffic violation detection system is realized in Python with OpenCV.
In this project, we will explore TensorFlow, and the Coco model for object detection. In it, I will describe the steps one has to take to load the pre-trained Coco model, how to use it, and how to build a simple implementation to detect objects from a given image. Moreover, besides presenting an example, I want to provide a small preface to what object detection is, explain what's behind the Coco SSD model, and introduce TensorFlow Object Detection API, the library initially used to train the model.
